A digital image is just a matrix of brightness values; an 8-bit image ranging from 0-255 for each pixel.

#### What is a color image?

A color image is a three channels, consisting of red, blue and green.

In a digital camera, each camera still records a single brightness value, but there is a red, green, or blue filter on each pixel. Afterwards, missing pixels in each channel are interpolated to form a complete image.

Human eyes are supposedly more sensitive to green, so a Bayer pattern has more green pixels than the others.

#### Demosaicing

There is only one color channel sampled at each pixel location, but then we need to interpolate the RGB values from the raw image.
**Bilinear interpolation**: use linear combination of neighboring pixels (an average of the neighbors)

Green channels:
Average of horizontal and vertical neighbors.

Red and blue channels:
Diagonals of the neighboring channels.
